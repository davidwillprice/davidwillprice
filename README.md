## 📖 Story

Having graduated with a Fine Art degree in 2014, I taught myself HTML and CSS to earn my first Web Developer & Designer position in 2015.

I've become development-focused over the last 11 years; I have pushed myself to learn new skills and work with new tech from JavaScript and React to Shopify and WordPress. I take pride in ensuring my work is responsive, accessible, modular, clean, and performant.
